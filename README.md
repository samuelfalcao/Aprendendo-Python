# Testes GUI

Pequeno projeto de exemplo com uma GUI usando `tkinter`.

Requisitos:
- Python 3.8+
- `pytest` (somente se for executar os testes)

Como executar a GUI:

```bash
python Testes.py
```

Como rodar os testes:

```bash
pytest
```

Observações:
- A função `validate_inputs(nome, idade_str, sexo)` possui testes em `tests/test_validation.py`.
- O arquivo principal `Testes.py` abre a interface gráfica quando executado diretamente.
